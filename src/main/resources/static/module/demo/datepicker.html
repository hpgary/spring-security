使用说明

描述：

日期选择与时间选择是基于 bootstrapDatePicker与bootstrapDatetimePicker进行指令化的。

使用：

							
html部分
参考示例

JS部分
$scope.myDate;

$scope.myDateTime;

$scope.callback = function(val, formatVal) {
	console.info(val + "-" + formatVal);
}

API说明：

参数 默认值 说明
options 配置属性
format yyyy-mm-dd 格式
setStartDate 开始时间
setEndDate 结束时间
callback 回调函数
