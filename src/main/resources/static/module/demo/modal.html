使用说明

描述：

模态框分为：确认框、警告框、消息框、错误框、自动关闭消息框、自动关闭警告框、自动关闭错误框，其中自动关闭是将属性autoClose设置为true。

确认框(confirm)：

						
cjhmeModal.confirm({
	title: '请确认',           //标题
	content: '您确定？',       //内容
	detail: '这里的详细内容！',  //详细内容
	ok: function() {         //确认按钮回调函数
		console.info("确认框 ok")
	},
	cancel: function() {    //取消按钮回调函数
		console.info("确认框 cancel")
	}
});

警告框(warning)：

						
cjhmeModal.warning({
	title: '警告框',           //标题
	content: '这是一个警告',    //内容
	ok: function() {         //确认按钮回调函数
		console.info("警告框 ok")
	}
});

警告框(warning)：

						
cjhmeModal.warning({
	title: '警告框',           //标题
	content: '这是一个警告',    //内容
	ok: function() {         //确认按钮回调函数
		console.info("警告框 ok")
	}
});

消息框(info)：

						
cjhmeModal.info({
	title: '消息框',
	content: '这是一个消息',
	ok: function() {
		console.info("消息框 ok")
	}
});

错误框(error)：

						
cjhmeModal.error({
	title: '错误框',
	content: '这是一个错误框',
	ok: function() {
		console.info("错误框 ok")
	}
});

自动关闭消息框(info)：

						
cjhmeModal.info({
	title: '自动关闭消息框',         //标题
	content: '这是一个自动关闭消息',  //内容
	autoClose: true              //是否自动关闭,值为true和false
});

自定义框：

						
cjhmeModal.customModal({
	templateUrl: "modalContent.html", //模版
	scope: $scope,                    //$scope
	size: 'md',                       //窗口大不
	controller: ['$scope', '$uibModalInstance', function($scope, $uibModalInstance) { //控制器
		$scope.ok=function(){   //控制器内部函数
			$scope.$dismiss();
		}
	}]
});
